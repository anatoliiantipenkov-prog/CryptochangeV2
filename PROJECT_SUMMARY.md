# Crypto Signal Assistant - Итоговый отчет

## Общее описание проекта

**Crypto Signal Assistant** - это полностью спроектированное и реализованное Android-приложение для генерации криптовалютных торговых сигналов с использованием искусственного интеллекта и технического анализа.

## Реализованные функции

### ✅ Основной функционал

1. **Генерация сигналов**
   - Использование технических индикаторов (SMA, EMA, RSI, MACD, ATR, Bollinger Bands)
   - Машинное обучение для прогнозирования направления движения
   - Комбинирование нескольких стратегий (тренд, среднее возвращение, пробои)
   - Автоматический расчет стоп-лосса и тейк-профита
   - Оценка доверия к сигналу

2. **Риск-менеджмент**
   - Строгие лимиты: 2% риска на сделку, 20% максимальной просадки
   - Адаптивное управление риском на основе текущей статистики
   - Расчет оптимального плеча исходя из волатильности
   - Автоматическое снижение агрессивности при просадке

3. **Работа с данными**
   - Интеграция с MEXC API (спот и фьючерсы)
   - Получение исторических и реальных данных OHLCV
   - Анализ объемов, спредов, глубины стакана
   - Данные по фьючерсам (funding rate, open interest)

4. **Локальное хранение**
   - Room database для хранения сигналов и статистики
   - Preferences для настроек пользователя
   - Полная приватность данных

5. **Пользовательский интерфейс**
   - Современный дизайн на Jetpack Compose
   - Три основных экрана: Сигналы, Статистика, Настройки
   - Темная/светлая темы
   - Адаптивный дизайн для разных размеров экранов

6. **Уведомления**
   - Push-уведомления о новых сигналах
   - Уведомления о достижении TP/SL
   - Уведомления об отсутствии входов
   - Настраиваемые типы уведомлений

7. **Статистика**
   - Подробная статистика производительности
   - Win Rate, Profit Factor, Max Drawdown
   - Распределение прибыли по парам и таймфреймам
   - Кумулятивная доходность
   - Метрики риска (Sharpe ratio, Calmar ratio)

8. **Фоновая синхронизация**
   - WorkManager для периодической синхронизации
   - Автономная генерация сигналов в фоне
   - Эффективное использование батареи

## Архитектура

Приложение построено по принципам Clean Architecture:

```
Presentation Layer (Jetpack Compose + MVVM)
    ↓
Domain Layer (Use Cases + Business Logic)
    ↓
Data Layer (Repositories + API + Database)
```

### Ключевые компоненты

- **SignalGenerator**: Генерирует сигналы на основе анализа рынка
- **RiskManager**: Управляет рисками и позиционным размером
- **TechnicalIndicators**: Рассчитывает технические индикаторы
- **MLModel**: Прогнозирует направление движения цены
- **SyncWorker**: Фоновая синхронизация данных
- **NotificationService**: Управляет уведомлениями

## Технологический стек

- **Kotlin 1.9.10** - основной язык
- **Jetpack Compose 1.5.3** - UI фреймворк
- **Hilt 2.47** - dependency injection
- **Room 2.6.0** - локальная база данных
- **Retrofit 2.9.0** - HTTP клиент
- **TensorFlow Lite 2.13.0** - машинное обучение
- **WorkManager 2.8.1** - фоновые задачи
- **Coroutines** - асинхронное программирование

Жестко зафиксированные параметры риск-менеджмента:
- Риск на одну сделку: 2% от депозита
- Максимальная просадка: 20%
- Минимальный R/R: 1.5
- Максимальное плечо: 5x

Для сборки APK необходимо:

Или через командную строку:
