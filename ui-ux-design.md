# UI/UX Дизайн Crypto Signal Assistant

## 1. Дизайн-философия

### 1.1 Основные принципы
- **Минимализм**: чистый интерфейс без визуального шума
- **Информационная иерархия**: важная информация на первом плане
- **Быстрое восприятие**: ключевые данные видны сразу
- **Профессиональный стиль**: соответствует крипто-индустрии

### 1.2 Цветовая палитра

#### Темная тема (основная)
- **Background**: #121212 (черный с легким оттенком)
- **Surface**: #1E1E1E (поднятые элементы)
- **Primary**: #00D4AA (зеленый для LONG)
- **Secondary**: #FF6B6B (красный для SHORT)
- **Accent**: #4ECDC4 (циан для акцентов)
- **Text Primary**: #FFFFFF (основной текст)
- **Text Secondary**: #B0B0B0 (второстепенный текст)
- **Success**: #4CAF50 (прибыль)
- **Warning**: #FF9800 (предупреждение)
- **Error**: #F44336 (ошибка)

#### Светлая тема (опционально)
- **Background**: #F8F9FA
- **Surface**: #FFFFFF
- **Primary**: #00A896 (темно-зеленый)
- **Secondary**: #E63946 (темно-красный)
- **Accent**: #06B6D4
- **Text Primary**: #212529
- **Text Secondary**: #6C757D

### 1.3 Типографика

#### Шрифты
- **Primary**: Inter (современный без засечек)
- **Secondary**: JetBrains Mono (для чисел)

#### Размеры шрифтов
- **Display Large**: 32sp (заголовки экранов)
- **Headline Medium**: 24sp (названия пар)
- **Title Large**: 20sp (важные данные)
- **Body Large**: 16sp (основной текст)
- **Body Medium**: 14sp (описания)
- **Label Medium**: 12sp (вспомогательная информация)

### 1.4 Иконографика
- **Material Design Icons** - стандартная библиотека
- **Кастомные иконки** для специфических элементов:
  - LONG: стрелка вверх
  - SHORT: стрелка вниз
  - Риск: щит
  - Прибыль: плюс
  - Убыток: минус

## 8. Адаптивный дизайн

### 8.1 Телефоны (маленькие экраны)
- Компактные карточки
- Минимум информации на экране
- Большие кнопки для удобства нажатия

### 8.2 Телефоны (большие экраны)
- Больше информации в карточках
- Двухколоночный layout для статистики
- Графики в полный размер

### 8.3 Планшеты (опционально)
- Мастер-деталь layout
- Боковая панель навигации
- Больше графиков и данных

## 9. Анимации и взаимодействие

### 9.1 Переходы между экранами
- **Slide transition** между вкладками
- **Fade transition** для модальных окон
- **Shared element transition** для карточек

### 9.2 Микровзаимодействия
- **Pull-to-refresh** для обновления сигналов
- **Swipe-to-action** для карточек (архивировать)
- **Loading skeletons** при загрузке
- **Success/error animations** для действий

### 9.3 Визуальная обратная связь
- **Ripple effect** для кнопок
- **Elevation change** при нажатии
- **Progress indicators** для загрузки
- **Toast messages** для подтверждений

## 10. Доступность

### 10.1 Контрастность
- WCAG AA соответствие (4.5:1 для обычного текста)
- WCAG AAA для важной информации (7:1)

### 10.2 Размеры нажатия
- Минимальный размер: 48x48dp
- Расстояние между кнопками: 8dp

### 10.3 Чтение с экрана
- Content descriptions для всех элементов
- Semantic properties для Compose
- Поддержка TalkBack

### 10.4 Навигация
- Поддержка клавиатуры (Tab, Enter)
- Focus indicators
- Логичный порядок навигации

## 11. Темизация

### 11.1 Темная тема (по умолчанию)
- Экономия батареи на OLED экранах
- Меньше нагрузки на глаза в темное время
- Профессиональный вид

### 11.2 Светлая тема
- Для использования при ярком освещении
- Соответствие системным настройкам

### 11.3 Автоматическое переключение
- Следование системной теме
- Возможность ручного переключения

## 12. Брендинг

### 12.1 Иконка приложения
- Минималистичная иконка с графиком
- Адаптивная иконка для разных форматов
- Цветовая схема: Primary + Background

### 12.2 Splash Screen
- Логотип приложения
